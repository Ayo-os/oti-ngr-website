# OTI NGR Ltd

Official website for OTI NGR Ltd — a Nigerian export company specializing in premium polypropylene (PP) resin for international markets.

## About

OTI NGR Ltd connects Nigerian petrochemical manufacturing with global demand, serving buyers across Turkey, Vietnam, Indonesia, India, and the MENA region. We are an NEPC-certified exporter with established banking relationships for secure international trade.

## Products

- Homopolymer Polypropylene (HS Code 3902.10)
- Multiple MFI grades available
- FOB Lagos shipping terms
- LC at Sight payment
